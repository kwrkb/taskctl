## 2026-07-15: git author を勝手に変更しない

- **何が起きたか**: グローバル設定の `user.name = yugosasaki` を「実名の漏洩リスク」と判断し、確認なしにリポジトリローカルで `kwrkb` へ変更してコミットした。実際は `yugosasaki` は意図的な開発用の名前だった。
- **ルール**: git の author 情報（user.name / user.email）は本人のアイデンティティ設定。「個人情報に注意」と言われても、**設定変更が必要かはユーザーに確認してから**行う。既存のグローバル設定は意図されたものとみなすのが既定。
